# Gitko Auto-Dispatch - Korean to English Keyword Mapping

This file documents the Korean keyword mappings used by the auto-dispatch system.
PowerShell script uses English patterns to avoid encoding issues.

## Lubit (Review & Documentation)

Korean → English

- 리뷰 → review
- 검토 → validate/check
- 확인 → verify/check
- 검증 → validate/audit
- 점검 → inspect
- 감사 → audit
- 문서화 → document
- 정리 → organize/document
- 보고서 → report
- 인수인계 → handoff

## Sian (Code Assist & Improvement)

Korean → English

- 제안 → suggest
- 개선 → improve/enhance
- 리팩터 → refactor
- 최적화 → optimize
- 성능 → performance
- 코드 어시스트 → code assist
- AI 제안 → AI suggest
- 자동 개선 → auto improve

## Parallel (Distributed Work)

Korean → English

- 병렬 → parallel
- 협업 → collaborate
- 분산 → distributed
- 전체 작업 → comprehensive
- 종합 → comprehensive
- 루빛과 시안 → both review and improve

## Gitko (Direct Implementation)

Korean → English

- 구현 → implement
- 수정 → fix/modify/update
- 추가 → add/create
- 삭제 → delete
- 버그 → bug
- 에러/오류 → error
- 배포 → deploy
- 빌드 → build
- 테스트 실행 → run test
- 스크립트 실행 → execute

## Usage Tips

When making requests in Korean, the system will match based on context.
For best results, include action verbs that clearly indicate the work type:

- "검토해줘" → Lubit
- "개선 제안" → Sian
- "리뷰하고 개선" → Parallel
- "구현해줘" → Gitko
