## 🛠️ 문제 해결

### 자동 진단
```powershell
.\troubleshoot.ps1
# 문제 자동 탐지

.\troubleshoot.ps1 -Fix
# 자동 수정 시도
```

### 수동 확인
```
1. Output Channel 확인
   View → Output → "Gitko Extension"

2. Health Check 실행
   Ctrl+Shift+P → "Gitko Dev: Health Check"

3. 설정 검증
   @gitko /check
```
