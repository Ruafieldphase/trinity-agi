# Verifying Lumen Releases

## Image Signature (cosign, keyless)
```bash
# Install cosign: https://docs.sigstore.dev/cosign/installation/
IMAGE="ghcr.io/<owner>/lumen-adaptive:<version>"
COSIGN_EXPERIMENTAL=1 cosign verify "$IMAGE"
```

## SBOM (SPDX)
- Release assets include `sbom.spdx.json` generated by Syft.
- Validate with tools like `spdx-sbom-tool`, `syft packages file:sbom.spdx.json`.

## Rekor Transparency Log
- Keyless signatures are recorded in Rekor.
- `cosign verify` output includes log index; view at https://search.sigstore.dev
