# 영어 동영상 자막 한글로 바꿔주는 프로그램 알려줘

영어 동영상의 자막을 한글로 번역해주는 프로그램과 서비스는 다양한 옵션이 있습니다. 각 도구의 특징과 사용 방법을 소개합니다.

## AI 번역 도구
- **DeepL Translate** : 전문적인 텍스트 번역에 뛰어난 성능을 보이는 AI 번역기로, 강의 콘텐츠 등 정확한 번역이 필요할 때 추천합니다.[1]
- **Transmonkey.ai** : OpenAI Whisper AI와 ChatGPT, Claude 등의 언어 모델을 활용해 정확한 번역을 제공하며, 130개 이상의 언어를 지원합니다. 동영상/오디오 파일 직접 처리 가능.[2]

## 자동 자막 생성 프로그램
- **Vrew** : 무료 프로그램으로 영상 업로드 시 자동으로 음성 인식하여 한국어 자막 생성. 추출 후 번역 가능.[1]
- **Subtitle Edit** : Whisper AI 통합으로 고정확 음성 인식 후 Google Translate API를 이용해 자동 번역. srt 파일 저장 지원.[3]

## 온라인 서비스
- **Happy Scribe** : 인간 번역가가 참여하는 프리미엄 서비스. 24시간 내 한국어 자막 생성 후 3~5일 이내 영어 번역 완료.[5]
- **Zeemo AI** : 모바일 앱 제공으로 편리한 사용. 동영상 업로드 시 자동 자막 생성 및 번역.[6]
- **Rev/Translated Labs** : AI 기반 빠른 번역 가능하지만 정확도 검토 필요.[4]

## 주요 특징 비교
| 도구          | 무료 여부 | 음성 인식 | 파일 형식 지원 | 번역 방식       |
|---------------|-----------|-----------|-----------------|------------------|
| Vrew          | 무료      | ✔️        | 동영상          | 수동 번역 필요  |
| Transmonkey.ai| 유료      | ✔️        | 동영상/SRT      | AI 자동 번역     |
| Happy Scribe  | 유료      | ✔️        | 다양한 형식     | 인간 번역       |

**간편 사용 추천**  
- 초보자: Vrew + DeepL 조합으로 무료 체험.[3][1]
- 고품질 요구: Happy Scribe의 인간 번역 서비스.[5]
- 실시간 처리: Zeemo AI 모바일 앱 활용.[6]

번역 후 자막 파일 형식 변환에는 **happyscribe Subtitle Converter**를 사용해 srt, vtt 등으로 변환할 수 있습니다.[1][5]
