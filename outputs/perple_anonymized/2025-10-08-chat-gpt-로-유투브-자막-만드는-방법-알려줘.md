You can create YouTube subtitles using ChatGPT by following these steps:

1. **Install a Chrome Extension**: Add the "Youtube Summary with ChatGPT" extension from the Chrome Web Store.[1][6][8]

2. **View Transcripts**: Once installed, open a YouTube video, and you'll see a "Transcript & Summary" button next to the video. Click it to view the full transcript generated by ChatGPT.[7][1]

3. **Generate Subtitles**: Use the extension to automatically generate subtitles, which you can then edit or translate as needed.[3][6]
