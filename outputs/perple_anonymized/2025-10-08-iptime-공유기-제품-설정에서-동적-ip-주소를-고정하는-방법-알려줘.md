# iptime 공유기 제품 설정에서 동적 ip 주소를 고정하는 방법 알려줘

iptime 공유기에서 동적 IP 주소를 고정 IP로 설정하려면 DHCP 서버의 IP 주소 예약 기능을 사용해야 합니다. 이 설정을 통해 특정 기기의 MAC 주소와 IP 주소를 고정적으로 매핑할 수 있으며, 네트워크 재연결 시에도 동일한 IP가 할당됩니다. 

## 설정 단계
1. **관리자 페이지 접속**  
   웹 브라우저에서 `192.168.0.1` 입력 → 기본 계정(`admin`/`admin`)으로 로그인.[1][2]
   *비밀번호를 변경한 경우 변경된 정보 사용*

2. **DHCP 설정 메뉴 이동**  
   - 관리 도구 → **고급 설정 → 네트워크 관리 → DHCP 서버 설정**[2][4]
   - "현재 사용 중인 IP 주소" 탭에서 연결된 기기 목록 확인[3][4]

3. **고정 IP 할당**  
   - IP를 고정할 기기 선택 → **수동 등록** 클릭[4]
   - MAC 주소와 고정할 IP 주소 입력[1][3]
     *예: NAS에 `192.168.0.5` 할당*

4. **IP 범위 분리**(권장)  
   | 구분 | IP 범위 | 용도 |
   |---|---|---|
   | 고정 IP | 192.168.0.2~192.168.0.9 | 서버, 프린터 등 고정 장치 |
   | 동적 IP | 192.168.0.10~192.168.0.254 | 일반 기기[5]  
   *이 설정은 DHCP 서버 설정에서 "IP 주소 대여 범위" 조정으로 가능*[5]

5. **적용 및 확인**  
   - 설정 저장 후 공유기 재부팅[1]
   - 대상 기기 재연결하여 IP 확인[2][4]

**주의사항**:  
- MAC 주소는 기기 네트워크 설정에서 확인 가능(윈도우: `ipconfig /all`, Android: Wi-Fi 상세 정보)[3][1]
- 중복 IP 방지를 위해 고정 IP는 동적 범위와 겹치지 않게 설정[5]

이 방법을 사용하면 NAS, IP 카메라, 네트워크 프린터 등 장치의 주소 변경 없이 안정적인 연결이 가능합니다.[4][2]
