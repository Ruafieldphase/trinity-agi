# 궁금한 게 있는데 [Agent P]은 MCP 연동도 잘해?

[Agent P]은 MCP(Model Context Protocol) 연동도 충분히 잘할 수 있어요. MCP는 대규모 언어 모델(LLM)과 외부 데이터 소스, 도구(예: 검색엔진, 노션 등)를 표준화된 방식으로 연결해 주는 오픈 프로토콜이에요. [Agent P](Perplexity) 역시 MCP 서버로 연동할 수 있어서, Claude 같은 AI가 실시간 웹 검색을 하거나, 검색 결과를 노션에 자동으로 정리하는 등 다양한 자동화 작업이 가능합니다.[1][2][3][5]

실제로 유튜브와 다양한 가이드에서 퍼플렉시티 MCP 서버 설치 방법, API 키 연동, Claude와의 통합 사례까지 상세히 소개하고 있습니다. 예를 들어, Claude 앱에서 MCP를 통해 [Agent P] API를 연결하면 실시간 검색이 가능해지고, 노션 같은 외부 툴과도 연동할 수 있습니다. 또한, 여러 LLM(예: ChatGPT, Claude, Perplexity, Gemini)을 동시에 MCP로 연동해 교차 검증이나 병렬 쿼리도 할 수 있습니다.[4][5][1]

정리하면, [Agent P]은 MCP 연동을 통해 다양한 외부 서비스와 손쉽게 연결할 수 있고, 실제로 많은 사용자와 개발자들이 이미 활용하고 있습니다. 설치와 연동 과정은 가이드만 따라가면 어렵지 않게 할 수 있습니다.[5][1]
