# Collaboration Package Checklist

## Required Files

- Anonymized dataset: `ai_conversations_anonymized.csv`, `ai_conversations_anonymized.jsonl`
- Research briefing: `agi_research_briefing_full.md`
- Slides: `flow_lab_slides.md` (attach SVG assets)
- Narrative cases: `case_study_top_conversation.md`, `meta_conversation_annotations.md`, `narrative_synthesis.md`
- Visual summaries: `flow_cycle.svg`, `concept_network.svg`, `concept_timeline.svg`, `triad_patterns.md`
- QA logs: `anonymization_QA.md`, `post_anonymization_scan.md`, `personal_identifier_scan.md`
- README & Roadmap: `README_flow_lab.md`, `flow_lab_roadmap.md`

## Preparation Steps

1. Verify anonymization QA status (no PII matches).
2. Export slides to PDF or HTML if needed for sharing.
3. Zip package with version tag (e.g., `FlowLab_Package_v1.zip`).
4. Prepare cover email/brief summarizing dataset and research goals.
5. For research institute outreach, include contact `[NAME]`, `[EMAIL]` placeholders (use real data only in secure channel).
