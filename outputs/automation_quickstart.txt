Flow Automation Quickstart
==========================

1. Flatten ChatGPT exports (Rua / Elro / Lumen)
   ```
   python d:\nas_backup\scripts\flatten_chatgpt_export.py ^
       --input ai_binoche_conversation_origin\rua\conversations.json ^
       --source rua
   ```
   Repeat for `--source elro` / `--source lumen` as needed.

2. Flatten Sena MCP conversations
   ```
   python d:\nas_backup\scripts\flatten_sena_export.py ^
       --input ai_binoche_conversation_origin\sena\conversations.json
   ```

3. Combine datasets + generate summary/weekday chart
   ```
   python d:\nas_backup\scripts\build_combined_conversations.py --sources rua sena elro lumen
   ```
   Outputs: `ai_conversations_combined.csv`, `ai_conversations_summary.txt`, `ai_conversations_weekday.svg`

4. Concept & metacognitive analytics
   ```
   python d:\nas_backup\scripts\build_concept_datasets.py
   python d:\nas_backup\scripts\build_metacognitive_markers.py
   python d:\nas_backup\meta_concept_alignment.py
   ```

5. Perple / Comet / Cople / Eru processing
   ```
   python d:\nas_backup\perple_flatten.py
   python d:\nas_backup\perple_anonymize.py
   python d:\nas_backup\scripts\build_perple_charts.py
   powershell -File d:\nas_backup\scripts\build_other_ai_index.ps1
   ```

6. QA and diagnostics
   ```
   python d:\nas_backup\scripts\scan_sensitive.py
   python d:\nas_backup\LLM_Unified\vertex_ai_test.py
   python d:\nas_backup\LLM_Unified\session_memory_monitor.py
   ```

7. (Optional) Run standard flow simulation
   ```
   python d:\nas_backup\lumen_auto_eval.py --runs 20 --cycles 5 --steps-per-phase 16 --export-csv d:\nas_backup\outputs\lumen_auto_eval_runs.csv
   ```

All helper scripts sit in `d:\nas_backup` and reuse existing outputs folders.
