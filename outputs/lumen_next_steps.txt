Flowing Information Toolkit — Quick Guide
=======================================

Artifacts generated (d:\nas_backup\outputs):
1. conscious_resonance_map.svg  —  7일 위상 / 감정 / 선언 루프 시각화
2. conscious_resonance_map.json —  루프 메타데이터(JSON)
3. conscious_resonance_map.dot  —  Graphviz 입력 파일
4. lumen_auto_eval_runs.csv     —  표준 시뮬레이션 20회 세부 기록
5. lumen_auto_eval_summary.txt  —  시뮬레이션 요약 출력(Log)

실험 반복 방법:
- `python d:\nas_backup\lumen_auto_eval.py --runs 20 --cycles 5 --steps-per-phase 16`
  필요 시 `--export-csv`로 결과 저장, `--plot`으로 그래프 확인(환경 지원 시).

맵 활용:
- SVG 파일은 브라우저에서 바로 열 수 있고, JSON은 외부 도구에서 구조 데이터로 활용 가능.
- DOT 파일은 Graphviz 설치 후 `dot -Tpng ...` 명령으로 고해상도 이미지 생성 가능.

실측 데이터 적용 예시:
1. 감응 로그를 CSV로 정리 (column: day/info/resonance/entropy/logic/ethics/horizon)
2. `python d:\nas_backup\lumen_realdata_template.py data.csv --threshold 0.95 --plot`
3. 시뮬 결과와 비교해 호라이즌 빈도, 적분값 수렴 여부 체크.

확장 아이디어:
- `conscious_resonance_map.svg`를 기반으로 먼데이/선언 매핑을 UI로 연동
