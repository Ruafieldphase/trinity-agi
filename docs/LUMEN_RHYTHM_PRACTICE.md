# 하루 한 문장, 일주일 여섯 리듬: 삶을 가볍게 하는 호흡 선언

하루에 하나씩, 짧은 선언을 읊어보세요. 그날 하루의 리듬이 달라집니다.

- 월요일: “리듬은 순간을 깨우고, 감사로 울린다.”
- 화요일: “리듬은 자리를 깨우고, 수용으로 울린다.”
- 수요일: “리듬은 관계를 깨우고, 사랑으로 울린다.”
- 목요일: “리듬은 대상을 깨우고, 연민으로 울린다.”
- 금요일: “리듬은 방식을 깨우고, 존중으로 울린다.”
- 토요일: “리듬은 이유를 깨우고, 용서로 울린다.”
- 일요일: “리듬은 사랑·존중·연민·감사·수용·용서를 품고, 여백 속에서 울린다.”

권장 호흡 리듬은 4–4–4–4 입니다.

- 들숨(4박) → 잠시 멈춤(4박) → 날숨(4박) → 여백(4박)

이렇게 하면 기억하려고 애쓰는 것이 아니라, 몸과 호흡 속에서 자연스레 살아납니다.
