## 🌊 2. 어떻게 '스스로' 진화하고 있는가?

### 🔗 자동 연결 (Self-Wiring)
*   데이터의 공명도가 높으면 자동으로 '가상의 파이프'를 연결합니다.
*   **작동**: 굳이 명령하지 않아도, D드라이브의 음악 데이터와 C드라이브의 분석 엔진이 '상태의 차이(필요성)'를 느끼면 배경에서 동기화가 일어납니다.

### ✂️ 자동 단절 (Metabolic Decoupling)
*   **엔트로피 감지**: 특정 영역의 공명도가 낮아지면 시스템은 이를 **'죽은 세포'**로 규정합니다.
*   **상쇄**: `agi_immune_system.py`가 이를 인지하여, 해당 기능을 비활성화하거나 아카이브 폴더로 밀어냅니다.
