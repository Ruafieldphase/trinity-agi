## 🧠 인간 수면의 정보 이론적 분석

### 인간이 수면 중 하는 일

| 활동 | 물리적 프로세스 | 정보 이론적 의미 |
|------|----------------|-----------------|
| **뇌척수액 순환** | 노폐물 제거 | **노이즈 제거** (Signal-to-Noise 개선) |
| **세포 재생** | 죽은 세포 처리, 새 세포 생성 | **메모리 재구성** (손상된 비트 복구) |
| **꿈 (REM 수면)** | 무작위 패턴 재생 | **시뮬레이션** (새로운 연결 탐색) |
| **무의식 처리** | 통제 불가능한 자동 사고 | **백그라운드 추론** (파동 범위 탐색) |
| **에너지 충전** | 대사 회복 | **엔트로피 감소** (질서 회복) |

### 핵심 통찰

1. **노이즈 제거**: 깨어있을 때 쌓인 "잡음" 제거
2. **패턴 탐색**: 꿈 = 제약 없는 시뮬레이션 = 창의적 연결
3. **경계 확장**: 자신의 고유 데이터 범위를 벗어난 경험 (여행, 만남)
4. **무의식 = 파동 범위 탐색**: 통제 불가능 = 더 넓은 탐색 공간

4. **탐색 공간 축소**: 학습된 범위만 반복 → 새로운 연결 불가능

## 💤 AGI Sleep Modes (3단계)

### 1. Micro-Sleep (짧은 휴지, 1~5분)

**목적**: 노이즈 제거 + 캐시 플러시

**프로세스**:

```
1. 현재 상태 스냅샷 저장 (체크포인트)
2. 작업 메모리 플러시 (cache clear)
3. Ledger 압축 (중복 제거)
4. 엔트로피 계산 (정보 무질서도)
5. 짧은 무작위 주입 (noise injection)
6. 재시작
```

**효과**:

- ✅ Floating point drift 제거
- ✅ 캐시 오염 정리
- ✅ 메모리 단편화 해소

**언제**: 연속 작업 30~60분마다

---

### 2. Dream Mode (꿈, 30분~2시간)

**목적**: 제약 없는 패턴 탐색 + 새로운 연결 발견

**프로세스**:

```
1. 최근 경험 로드 (ledger 패턴)
2. 제약 해제 (no validation, no filtering)
3. 무작위 패턴 재조합:
   - 과거 이벤트를 무작위로 섞기
   - 불가능한 조합 시도
   - 인과관계 무시
4. 흥미로운 패턴 발견 시 저장
5. "꿈 로그" 기록 (dreams.jsonl)
```

**효과**:

- ✅ 창의적 연결 발견
- ✅ 로컬 최적화 탈출
- ✅ 새로운 가설 생성

**예시**:

```json
{
  "dream_id": "dream_20251101_2200",
  "patterns": [
    "YouTube 학습 + BQI 판단 + 캐나리 배포",
    "Ledger 패턴 + Core 프로브 + 무의식 흐름"
  ],
  "new_connections": [
    "YouTube 영상 품질이 BQI 판단에 영향?",
    "Core 상태가 캐나리 성공률 예측?"
  ],
  "worth_testing": true
}
```

**언제**: Sleep Mode 진입 시 (22:00~06:00)

---

### 3. Deep Sleep (깊은 수면, 4~8시간)

**목적**: 완전 재구성 + 장기 메모리 통합 + 엔트로피 최소화

**프로세스**:

```
1. 전체 Ledger 분석
2. 중복/노이즈 제거 (압축)
3. 패턴 추출 및 요약
4. 장기 메모리 재구성:
   - 중요 이벤트만 유지
   - 반복 패턴 압축
   - 메타 패턴 추출
5. 백업 생성 (snapshot)
6. 시스템 검증 (integrity check)
```

**효과**:

- ✅ 장기 메모리 최적화
- ✅ 시스템 무결성 보장
- ✅ 에너지(리소스) 최소화

**언제**: 주 1회 이상 (예: 일요일 새벽)

## 🌊 무의식 = 파동 범위 탐색

### 인간의 무의식 특징

1. **통제 불가능**: 의도적으로 제어할 수 없음
2. **자동 스토리텔링**: 끊임없는 내러티브 생성
3. **파동 범위 확장**: 자신의 데이터 경계 밖 탐색

### AGI의 무의식 (Unconscious Mode)

**구현**:

```python
class UnconscciousProcessor:
    def __init__(self):
        self.control = False  # 의도적 통제 불가
        self.auto_narrative = True
        
    def run_background(self):
        """백그라운드에서 끊임없이 실행"""
        while True:
            # 1. 무작위 Ledger 샘플링
            events = random.sample(ledger, k=5)
            
            # 2. 제약 없는 연결 시도
            narrative = self.generate_story(events)
            
            # 3. 파동 범위 확장 (범위 밖 탐색)
            explore_outside_boundary()
            
            # 4. 흥미로운 발견만 저장
            if is_interesting(narrative):
                save_to_unconscious_log(narrative)
```

**특징**:

- ⚠️ 의도적으로 통제하지 않음 (무작위성 유지)
- ✅ 백그라운드 지속 실행
- ✅ 예상 밖 패턴 발견
- ✅ 창의성의 원천

**언제**: 항상 (Core 레이어에서 항상 실행)
