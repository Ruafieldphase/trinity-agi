# 🎯 Comet: 빠른 Extension 설치 (1분)

## 현재 상태

✅ HTML 버전이 정상 작동 중입니다!

- 응답 속도: 4초
- 처리 완료: 12개 작업

## 🚀 Extension 업그레이드 (선택사항)

### 왜 업그레이드?

- 🌙 탭 닫아도 백그라운드 실행
- 🔄 브라우저 재시작 시 자동 재개
- 💾 통계 영구 저장
- 🔔 데스크톱 알림

### 3단계 설치

#### 1. Chrome 익스텐션 페이지

```
chrome://extensions/
```

(주소창에 복붙)

#### 2. 개발자 모드 ON

우측 상단 토글 클릭 → 파란색으로

#### 3. 폴더 로드

"압축해제된 확장 프로그램 로드" 클릭
→ 이 폴더 선택:

```
d:\nas_backup\chrome_extension\comet_worker
```

### 설치 후 확인

✅ "Comet Worker v2.0.0" 표시
✅ 툴바에 아이콘 나타남

### 시작하기

1. 아이콘 클릭
2. "🚀 작업 처리 시작" 클릭
3. 팝업 닫기 (백그라운드 계속 실행!)

## 💡 참고

- HTML 버전도 계속 사용 가능
- Extension은 더 안정적인 장기 실행용
- 둘 다 같은 API 사용 (충돌 없음)
