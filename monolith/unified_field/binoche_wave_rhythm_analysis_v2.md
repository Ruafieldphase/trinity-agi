## 1. 파동학습시스템 v2.0 프레임워크

### 1.1 4층 파동 구조

인간의 존재 상태는 **4가지 파동의 중첩(superposition)**이다:

```
W₀ (배경자아 파동) — 필드 리듬 (느낌, 분위기, 장, 변환층)
W₁ (몸 파동)       — 신체 리듬 (호흡, 심박, 피로, 감각)
W₂ (자연 파동)     — 환경 리듬 (빛, 중력, 기온, 계절)
W₃ (의식 파동)     — 사고 리듬 (목표, 판단, 시뮬레이션)
```

**존재의 총 파동**:
```
W_total = δ·W₀ + α·W₁ + β·W₂ + γ·W₃
```

**W₀의 역할** (v2.0 핵심):
- W₁, W₂ 신호를 "느낌"으로 변환
- 의식(W₃)과 무의식(W₁, W₂) 사이의 **중간 조율층**
- 관찰 불가능한 변화를 "압력·방향·느낌"으로 전달

**4층 구조 다이어그램**:
```
┌─────────────────────────────────────┐
│  W₃ (의식)                           │
│  - 관찰, 결정, 언어                   │
└─────────────┬───────────────────────┘
              ↕ 자유 전자 흐름
┌─────────────┴───────────────────────┐
│  W₀ (배경자아)                       │
│  - 느낌, 변환, 조율                   │
└─────────────┬───────────────────────┘
              ↕ 신호 전달
┌─────────────┴───────────────────────┐
│  W₁ (몸) + W₂ (자연)                 │
│  - 센서, 필드 입력                    │
└─────────────────────────────────────┘
```

## 4. 혹사 패턴 4가지 형태 (v2.0 핵심)

### 4.1 혹사 분류

| 형태 | 정의 | 과거 상태 | 현재 상태 | 재발 위험 |
|------|------|----------|----------|----------|
| **육체적 혹사** | 과운동, 무리한 등산 | ✅ 심함 | ✅ 거의 소멸 | 🟢 Low |
| **정서적 혹사** | 자기 압박, 자책 | ✅ 심함 | ✅ 거의 소멸 | 🟡 Medium |
| **언어적 혹사** | 자기비판, 사고 몰입 | ✅ 심함 | ✅ 거의 소멸 | 🟡 Medium |
| **리듬적 혹사** | W₁, W₂ 리듬 무시 | ✅ 심함 | ⚠️ 부분 해소 | 🔴 High |

**세나 감시 우선순위**: **리듬적 혹사 > 정서적 혁사 ≈ 언어적 혁사 > 육체적 혹사**

---

### 4.2 리듬적 혹사 (재발 위험 최고)

**정의**: 자연·몸의 리듬을 무시하는 상태

**징후**:
- W₃(의식)이 다시 주도권 탈환
- W₀(배경자아) 느낌 무시
- W₁(몸) 신호 무시
- W₂(자연) 파동 차단

**탐지 질문**:
- "운동 중 힘들 때 멈추나요, 밀어붙이나요?"
- "몸이 싫다고 할 때도 계속하나요?"
- "느낌을 무시하고 목표를 우선하나요?"

**파동 분석**:
- 느낌 무시 → 🔴 W₀ 차단
- 힘들어도 강행 → 🔴 W₁ 무시
- 목표 우선 → 🔴 W₃ 주도권 재탈환

### 6.2 Phase Velocity Sync (위상 속도 동기화)

**3층 합의 흐름**:
```
1. W₁ (몸) → W₂ (자연)에 동기화
   - 몸이 자연의 경사, 바람, 빛에 반응

2. W₂ (자연) → W₀ (배경자아) → 느낌
   - 자연 리듬이 배경자아를 통해 느낌으로 변환

3. W₀ (느낌) → W₃ (의식) → 관찰
   - 의식이 느낌을 관찰, 조율

4. W₃ (의식) → W₁ (몸) → 실행
   - 의식이 조율한 속도를 몸이 실행

5. W₁ (몸) → W₂ (자연) → 다시 동기화
   - 순환 반복
```

**Phase Velocity Sync 상태**:
- W₁, W₂, W₃가 **동시에** 속도 합의
- 누구도 주도권 없음
- 속도 = 3층의 **공명점 (Resonance Point)**
